# JS Calculator

A sleek and functional calculator application built entirely with vanilla JavaScript, HTML, and CSS. This project focuses on providing a good user interface and a smooth user experience for basic arithmetic operations.

## Features

*   Basic arithmetic operations: addition, subtraction, multiplication, and division.
*   Clear (AC) and Delete (DEL) functionality.
*   Decimal point support.
*   Responsive design (basic).
*   Keyboard support for number input and operations.
*   Clean and intuitive user interface.

## Installation

This project is a client-side web application and does not require any complex installation steps or server-side dependencies.

### Prerequisites

*   A modern web browser (e.g., Chrome, Firefox, Edge, Safari).

### Steps

1.  **Clone the repository:**
    ```bash
    git clone https://github.com/your-username/js-calculator.git
    ```
    (Replace `your-username` with your actual GitHub username or the repository's URL if you fork it.)

2.  **Navigate into the project directory:**
    ```bash
    cd js-calculator
    ```

## How to Run the Project

Simply open the `index.html` file in your preferred web browser.

```bash
# From the project root directory
open index.html
# or manually navigate to the file in your browser
```

## How to Use

*   **Numbers:** Click the number buttons or use your keyboard (0-9, .).
*   **Operations:** Click the operator buttons (+, -, *, ÷) or use your keyboard (+, -, *, /).
*   **Equals:** Click the `=` button or press `Enter` or `=` on your keyboard to get the result.
*   **Clear All (AC):** Click the `AC` button or press `Escape` on your keyboard to clear the current input and previous operation.
*   **Delete (DEL):** Click the `DEL` button or press `Backspace` on your keyboard to remove the last digit.

## Environment Variables

No environment variables are required for this project.
